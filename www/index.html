Welcome to pandapache

Installation

To install pandapache on your system, follow these simple steps:

  1. Download the latest version of pandapache from the official website.
  2. Extract the downloaded files to a directory on your computer.
  3. Open a terminal window and navigate to the directory where you extracted the files.
  4. Run the installation command: ./install.sh
  5. Follow the on-screen instructions to complete the installation process.

Configuration

Once pandapache is installed, you can configure it according to your requirements:

  • Open the configuration file located at /etc/pandapache.conf.
  • Adjust the settings as needed, such as specifying the root directory for your websites.
  • Save the changes and exit the text editor.
  • Restart the pandapache service to apply the new configuration.

Usage

With pandapache installed and configured, you can start hosting your static websites:

  1. Create a new directory for your website files.
  2. Place your HTML, CSS, and JavaScript files in the website directory.
  3. Start the pandapache service using the command: sudo systemctl start pandapache
  4. Open a web browser and navigate to http://localhost to view your website.
